## 🚀 Quick Start

```bash
# 1. Install
npm install

# 2. Create .env
VITE_API_BASE_URL=http://localhost:8000

# 3. Run
npm run dev

# 4. Open
http://localhost:5173/login

# 5. Login with
Email: john.doe@acme-corp.com
Password: password123
```
